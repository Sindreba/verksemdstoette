Phishing

Phishing

Phishing er og fiske etter sensetiv informasjon f.eks passord eller kredittkkortnummer på nettet.
Dette kan skje via epost, der dei sender deg eit falsk skjema og ber deg registrera sensitiv informasjon,
som, brukarnavn og passord på ulikenettsidar dei veit du har, eller bankkonto, kredittkort osv.
Dei sender ofte ein link til f.eks nettbanken din, berre at dette er noko dei har laga og ser ut som ein
eksakt kopi av den du har, det er då du må sjå på url navnter, fordi dei kan ikkje ha eksakt samme url navn,
som det banken din har, eller du kan sjå på eposten du fekk.
Dei som driver med Phishing er som oftast ute etter pengar.
Spearphishing eller spydfisking er ein målretta for for nettfisking, dei går etter enkeltpersonar,
dei samlar inn informasjon på førehand, f.eks kva bank du brukar sånn at dei kan kopiere nettsida deirans.
Whaling eller kvalfangst er at dei går rett mot direktøren eller personar i leiarstillingar.

ID Tjuveri

ID Tjuveri

Dei vill svindle deg for varer eller tenester, den vanlegaste formen for id tjuveri er når nokon
vill skaffa seg ein økonomisk fordel i andres navn, som f.eks ta ut lån, kreditt, gods og tjenester i andres navn.
For å redusera risikoen for å bli svindla seier NorSIS at du bør, ha ein rutine for å sjekka legitimasjon,
Før dei gjer avtalar med ny kundar dei ikkje kjenner til.
Bruka ein form for sikker elektronisk ID som signatur frå kundar som bestillar på internett, for eksempel BankID.
Kryptere viktig informasjon som blir utveksla med kundar via nettsider eller e-post.
Dei burde ikkje samla inn eller registrera meir informasjon enn det som er nødvendig for å ta hand om kundeforholdet.
Regelmessig sletta kundeinformasjon ein ikkje trenger lengre og berre gje tilsette som trenger det,
tilgang til kundeinformasjonø.

Hacking

Hacking

Ein Hacker eller datasnok som det heiter på norsk, er ein person som setter pris på intellektuelle utfordringar,
ved å bryta grensar eller jobba seg rundt begrensingar på eit felt han/ho er interisert i.
Ein hacker er ein person som likar å utforska detalj i eigenskaper ved programmbare systemar, i motsetning,
til vanlege brukare som foretrekkar å bare læra seg det mest nødvendiga.
Ein person som veit å setta pris på "hack value"
Ein person som er god til å programmera raskt.
Ein ekspert på eller entusiast på vilkårlige områder.
Ein som settar pris på den intellektuelle utfordringa det er å bryta frensar eller jobba seg rundt hindringar.
